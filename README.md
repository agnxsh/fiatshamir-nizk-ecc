## Fiat Shamir Non-Interactive Zero Knowledge Proof with an Elliptic Curve Implementation

Curve used here is Edwards25519

To run the project

Run this command:

```
$ go build
```

For Docker image

Run this command

```
$ docker compose up -d
```
